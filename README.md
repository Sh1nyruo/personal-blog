# 学无止境——个人博客网站
## 项目介绍
这个repo包含了我搭建的个人博客网站。

这个项目前端使用了[cwarcup.com](https://github.com/Cwarcup/personal-blog)提供的模板，使用Next.js和Tailwind CSS搭建。

后端使用了Django REST Framework。
